04

Usage Rules

Guidelines for keeping the brand consistent across all projects and contexts.

Do

  • Use --color-accent for all primary CTAs, active states, and key links
  • Use Raleway 700 exclusively for headings (h1–h3)
  • Maintain the three background depth levels (base → surface → elevated) for visual hierarchy
  • Keep body text at Poppins 300/400 — never heavier than 400 for paragraphs
  • Use the < Project /> code-tag format for all project titles
  • Apply --color-border-accent for borders on accent-highlighted cards
  • Use CSS variables — never hardcode hex values in component files

Don't

  • Don't use white or light backgrounds — all surfaces must be warm dark
  • Don't add a second accent color; gold is the only accent
  • Don't use font-weight 600 or 700 in body text (Poppins only)
  • Don't use blue as a link color — accent gold covers all interactive elements
  • Don't mix Inter or other sans-serifs with Poppins in the same interface
  • Don't use drop shadows for decoration — only for elevation signals
  • Don't use the school-blue (#004aad) anywhere — it's been replaced by the gold accent
05

CSS Variables Reference

Copy this :root block into any project's main CSS file. For Tailwind, map these values into tailwind.config.js under theme.extend.colors.

/* Refined Ember — Aleksei Lopatin brand system v1.0 */ :root { /* Backgrounds */ --color-base: #141210; --color-surface: #201c18; --color-elevated: #2c2620; /* Accent (gold) */ --color-accent: #d4a44b; --color-accent-hover: #e8b85c; --color-accent-muted: rgba(212, 164, 75, 0.15); --color-on-accent: #141210; /* Text */ --color-text: #e8dcc8; --color-text-muted: #9a8870; --color-text-faint: #5a4e42; /* Borders */ --color-border: #3a3020; --color-border-accent: rgba(212, 164, 75, 0.25); --color-border-strong: #5a4e42; /* Semantic */ --color-success: #7c9a52; --color-error: #b85042; --color-warning: #d4a44b; /* Typography */ --font-heading: 'Raleway', sans-serif; --font-body: 'Poppins', sans-serif; --font-mono: 'Courier New', Courier, monospace; /* Spacing (rem, base: html font-size 62.5%) */ --space-xs: 0.4rem; --space-sm: 0.8rem; --space-md: 1.6rem; --space-lg: 2.4rem; --space-xl: 4.0rem; --space-2xl: 6.4rem; /* Border radius */ --radius-sm: 3px; --radius-md: 6px; --radius-lg: 10px; /* Shadows */ --shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.4); --shadow-md: 0 4px 12px rgba(0, 0, 0, 0.5); /* Transition */ --transition: 200ms ease; }

For Tailwind CSS projects (school portal, gradebook), add to tailwind.config.js:

module.exports = { theme: { extend: { colors: { ember: { base: '#141210', surface: '#201c18', elevated:'#2c2620', accent: '#d4a44b', text: '#e8dcc8', muted: '#9a8870', border: '#3a3020', success: '#7c9a52', error: '#b85042', } }, fontFamily: { heading: ['Raleway', 'sans-serif'], body: ['Poppins', 'sans-serif'], } } } }
